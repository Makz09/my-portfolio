## 🛠️ Local Development

### Installation
1.  Clone the repository:
    ```bash
    git clone https://github.com/Makz09/portfolio.git
    ```
2.  Navigate to the webapp folder:
    ```bash
    cd webapp
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```

### Running Locally
```bash
npm run dev
```

### Deployment
```bash
npm run deploy
```
